# 前端页面与国际化设计文档

> **创建日期：** 2026-03-05
> **状态：** 已批准
> **风格：** 极简学术科技 + 论文纸黄棕色调
> **动效策略：** 混合方式（关键操作明显反馈，常规交互克制优雅）

---

## 1. 视觉系统

### 色彩系统

**主色调：**
- 背景：米白色 `#faf8f3`
- 论文纸色：淡黄棕 `#f4f1e8`
- 深色文字：墨黑 `#2d2a24`

**强调色：** 学术棕 `#8b6f47`

**辅助色：**
- 成功：橄榄绿 `#6b8e23`
- 警告：琥珀棕 `#d4a574`
- 错误：深红棕 `#a0522d`
- 处理中：渐变棕金 `#8b6f47 → #c9a961`

**中性色：**
- 浅灰棕 `#e8e4db`
- 中灰棕 `#c4bfb3`
- 深灰棕 `#6b6560`

### 字体系统

- 标题/数字：SF Pro Display / PingFang SC
- 正文：-apple-system 系统字体栈
- 代码/数据：SF Mono
- 学术引用：Georgia / 宋体（衬线）

### 视觉效果

- 磨砂纸质效果（backdrop-filter: blur(20px) + 纸纹理）
- 微妙阴影（0 2px 16px rgba(45,42,36,0.08)）
- 圆角：12px（卡片）、8px（按钮）、24px（对话框）
- 边框：1px solid rgba(139,111,71,0.15)
- 纸张纹理：细微噪点叠加（opacity: 0.03）

## 2. 组件设计

### 论文卡片（Paper Card）

- 背景：论文纸色 + 磨砂效果
- 悬停：translateY(-4px) + 阴影加深
- 布局：左侧 PDF 图标 | 中间标题+元信息 | 右侧状态+操作
- 状态指示器：左侧 4px 彩色边框

### 进度条（Progress）

- 背景：浅灰棕 `#e8e4db`
- 填充：渐变棕金 `#8b6f47 → #c9a961`
- 流动光泽效果（shimmer animation）
- 高度 8px，圆角 4px

### 状态标签（Badge）

- 圆角胶囊形状（border-radius: 12px）
- 半透明背景 + 对应颜色边框
- 处理中状态：脉冲动画（pulse）
- 字体 12px，字重 600

### 按钮系统

- 主按钮：学术棕背景 + 白色文字 + 微妙阴影
- 次要按钮：透明背景 + 棕色边框 + 棕色文字
- 危险按钮：深红棕背景 + 白色文字
- 悬停：scale(1.02) + 阴影加深
- 点击：scale(0.98) 反馈

### 上传区域

- 虚线边框（2px dashed #c4bfb3）
- 拖拽悬停：实线 + 背景高亮（#8b6f47 10% opacity）
- 中央大尺寸上传图标 + 拖拽时放大动画

## 3. 交互与动效

### 页面加载（stagger animation）

- 导航栏：0ms → 标题：100ms → 卡片：每个延迟 80ms
- opacity + translateY(20px)
- 缓动：cubic-bezier(0.16, 1, 0.3, 1)

### 关键操作动效（明显反馈）

**上传过程：**
- 上传中：进度条流动光泽 + 百分比数字滚动
- 完成：成功图标弹出（scale + rotate）+ 绿色波纹

**处理状态变化：**
- 状态标签：颜色渐变过渡（500ms）
- 进度条：平滑填充 + 光泽移动
- 卡片边框：颜色呼吸动画（pulse）

**删除操作：**
- 确认对话框：从底部滑入
- 删除中：卡片淡出 + 缩小（scale: 0.95）
- 完成：卡片向左滑出 + 其他卡片平滑上移

### 常规交互（克制优雅）

- 卡片悬停：translateY(-4px), 300ms
- 按钮悬停：背景色加深, 200ms
- 链接：下划线滑入 scaleX, 250ms
- 按钮点击：scale(0.98), 100ms

### SSE 实时更新动效

- 新状态到达：卡片轻微闪烁（highlight flash）
- 进度更新：数字滚动动画
- 完成通知：Toast 从右上角滑入 + 弹性效果

## 4. 页面布局

### 论文列表页 (`/papers`)

- 顶部固定导航（backdrop-blur）
- 页面标题 + 状态筛选 Tab（下划线滑动动画）+ 上传按钮
- 论文卡片列表（每页 20 条）
- 空状态：优雅插图 + 引导文案
- 骨架屏加载（skeleton loading）
- 分页控件

### 论文详情页 (`/papers/:id`)

- 面包屑导航
- 左侧固定宽度（360px）：论文信息卡片 + 进度条 + 操作按钮
- 右侧自适应：论文总结（可折叠 Accordion）+ 思维导图（点击放大 Lightbox）
- 移动端：垂直堆叠布局

### 积分历史页 (`/credits`)

- 余额卡片：大字号数字 + 微妙动画
- 交易历史表格（斑马纹）
- 金额：正数橄榄绿，负数深红棕
- 分页控件
- 移动端：转为卡片列表

## 5. 上传对话框

### Tab 1: 文件上传

- 拖拽区域：虚线边框 → 拖拽时实线高亮
- 文件选中：显示文件名 + 大小 + 预览图标
- 积分检查：不足时禁用按钮 + 红色提示
- 上传进度：进度条 + 百分比

### Tab 2: arXiv 链接

- 输入框：arXiv URL
- 格式提示：支持 abs 和 pdf 两种格式
- 积分检查：同上

### 通用

- Tab 切换：下划线滑动动画
- 底部：当前积分 + 本次消耗 + 操作按钮

## 6. 国际化（Paraglide）

### 支持语言

- 简体中文（zh-CN）
- English（en）

### 语言切换器

- 位置：导航栏右上角（用户头像旁）
- 样式：下拉菜单
- 持久化：localStorage

### 翻译范围

**UI 文本：** 按钮、标题、表单标签、提示信息

**状态文本：**
| 中文 | English |
|------|---------|
| 等待处理 | Pending |
| 提取文本中 | Extracting Text |
| 生成图片中 | Generating Image |
| 已完成 | Completed |
| 处理失败 | Failed |

**错误消息：**
| 中文 | English |
|------|---------|
| 积分不足 | Insufficient credits |
| 文件过大 | File too large |
| 仅支持 PDF 文件 | Only PDF files allowed |
| 无效的 arXiv 链接 | Invalid arXiv URL |

**交易类型：**
| 中文 | English |
|------|---------|
| 注册赠送 | Initial Credits |
| 处理论文 | Paper Processing |
| 退款 | Refund |
| 购买积分 | Purchase Credits |

### 翻译策略

- UI 文本：完全翻译
- 论文标题/内容：保持原文
- 错误消息：翻译 + 保留错误代码
- 日期时间：本地化格式
